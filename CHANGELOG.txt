Made I.T. Theme Changelog
==========================

Version 3.0.1 - 2026-06-03
-------------------------
Added
- New Git changelog and release tooling with `.github/skills/project-git-change-log/SKILL.md`
- New CLI tooling in `bin/` for building node subprojects and exporting WordPress filters/hooks
- New theme screenshot image `screenshot.png`

- Updated theme assets including `assets/bootstrap-5/style.css`, `assets/css/gutenberg.css`, and `style.css`

- Updated `inc/security/modules/Honeypot.php` to add honeypot support for spam mitigation
- Updated pattern `patterns/portfolio_wireframe.php`

Fixed
- Fixed build tooling consistency and package metadata for Gutenberg block packages
- Cleaned up generated source maps and build artifacts for updated block builds

Removed
- Removed obsolete Gutenberg content column build source maps and CSS map artifacts from the repository

Notes
- This release is primarily a tooling and Gutenberg block maintenance update, with improved build automation, editor integration, and security support.
